# Student Marks Analyzer (NumPy)

A beginner-friendly Python project that demonstrates the power of **NumPy** for performing data analysis on student marks.

## Features

* Display all student marks
* Find the highest and lowest marks
* Calculate the average marks
* Display students who passed (marks ≥ 40)
* Display students scoring above 80
* Add grace marks to every student's score
* Reshape the marks into a 2D array

## Technologies Used

* Python
* NumPy

## Concepts Practiced

* NumPy Arrays
* Array Operations
* Boolean Indexing
* Statistical Functions (`max()`, `min()`, `sum()`, `mean()`)
* Array Reshaping (`reshape()`)

## How to Run

1. Install NumPy:

   ```bash
   pip install numpy
   ```

2. Run the program:

   ```bash
   python student_marks_analyzer.py
   ```
